Recommended Style

To help make the content as easy as possible to grasp, we are using highlighted and bold text as follows:

  • Use the highlight tag for any actions you want the user to take. Example: "To open a session file, select File → Open..."
  • Use bold text for any other item you want to emphasize that is not an action. Example: "...data is visible in the Node Table".

Cytoscape Tutorial Development

Welcome to a tutorial on how to build a Cytoscape Reveal tutorial!

If you are interested in preparing training materials for Cytoscape Apps, you have come to the right place! This tutorial will guide you through how to prepare a tutorial for a Cytoscape App.

Let us know if you have an questions or suggestions.

Setup

  • Create a GitHub account if you don't already have one
  • Fork the Cytoscape-tutorials repo
  • Consider installing an html editor, for example Brackets, an open source code editor.

Navigating Materials

Let's start by reviewing the latest Cytoscape training material to get familiar with navigation.

  • Presentations, Protocols and Modules
    • - browse through the current materials and practice reveal's unique navigation system
  • Notice how the navigation goes left-to-right and top-to-bottom
  • To get an overview of the entire presentation, click Esc
  • To display speaker notes and a timer, click S

A helpful graphic explaining the reveal navigation is available in our README.

Navigating Materials

As you saw from reviewing the Cytoscape training material, it consists of Presentations and Protocols, and Modules. In reveal, a presentation or protocol can be composed of a combination of slides and modules.

Tutorials focused on Cytoscape Apps, or Cytoscape workflows, are designed as Protocol Modules.

Navigating Materials

Let's start exploring the structure of the repo in your favorite html editor:

  • Navigate to the app module template in protocols/modules/module-app. Each module contains both index.html and module.html. The latter contains the code representing slides and content, the former is simply a wrapper to view the module on its own.
  • Note that the index file points to the custom Cytoscape protocol css. This style is described in detail in the README.

Creating a Tutorial

  • If you don't have it open already, open protocols/module/template_app/module.html in your favorite html editor.
  • To start building your protocol using this template, save a copy of it in a new directory under protocols/modules named after your app or workflow.

    • You're ready to go!!

Creating a Tutorial

  • When your tutorial is ready, make a pull request to the cytoscape-tutorial repo to add your app tutorial to the growing collection shared with all Cytoscape users.
  • Share your completed protocol on the Cytoscape App Dev List.
